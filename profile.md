# 学员画像

- **AI 基础**：有基础 — 熟悉 LLM、Prompt 工程、Agent 架构、Tool Use、Workflow
- **Web3 基础**：有基础 — 了解钱包、交易、Gas、智能合约、链上数据读取
- **编程能力**：会基础脚本 — Python 为主，能写自动化脚本和 API 调用
- **目标方向**：产品研究（Web3 产品分析 / AI+Web3 应用场景）
- **每日可投入时间**：3 小时
- **输出语言**：中文为主，英文为辅

## 学习目标

1. **深入理解 AI × Web3 交叉领域的产品逻辑** — 不只停留在概念，能拆解真实产品的架构和设计决策
2. **建立系统化的链上 Agent 产品分析框架** — 从输入层（Chain-aware Context）到执行层（Tool Use）、流程层（Workflow）、权限层（Wallet），逐层分析产品是否"够硬"
3. **输出高质量产品研究报告** — 选定 2-3 个赛道，产出有洞察的分析报告
4. **完成 Hackathon 项目实践** — 将产品研究落地为可演示的最小可行产品

## 学习风格

- 偏向动手实践 — 边学边做实验和 demo
- 喜欢用类比和对比来理解新概念
- 重视"为什么"而非"是什么"
